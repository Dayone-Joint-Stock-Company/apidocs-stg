​Welcome to Got It API Documentation

Đây là website chứa các thông tin cần thiết giúp các đối tác của Got It tích hợp với các dịch vụ do Got It cung cấp. Những mục chính như sau:

  • Thông tin chung về cách tích hợp Got It API, bảng các mã lỗi
  • Thông tin về các flow, các quy định khi sử dụng Got It API
  • Thông tin đặc tả cho từng bộ API
    • Bộ Biz API: dùng để phát hành voucher Got It
    • Bộ Merchant API: dùng để kiểm tra và đánh dấu sử dụng voucher Got It
    • Bộ RewardHub API: dùng để tích hợp hệ thống quà tặng
